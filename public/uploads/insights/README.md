# Insight featured images

Staff upload featured images for articles from the dashboard (Insights → Add article / Edit article). Files are stored here (or in Vercel Blob when deployed) and the path is saved in the database, similar to applicant CVs.

- Allowed: JPEG, PNG, WebP, GIF. Max 4MB.
- The optional “Image title” is stored in the `Insight.imageTitle` field for accessibility or display.
