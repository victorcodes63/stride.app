## 30-minute demo script

### 0. Opening — set the scene (2 min)

**Say:**  
*“SwiftFreight is a regional cargo operator — drivers on the Mombasa corridor, warehouse teams at Industrial Area, dispatch control in Nairobi, and a customs desk. They run payroll, shifts, and compliance on Imara instead of spreadsheets and WhatsApp.”*

**Show:** Dashboard overview → [https://hcm.raventechgroup.com/dashboard](https://hcm.raventechgroup.com/dashboard)

**Point out KPIs:** active headcount, pending leave, credentials expiring, attendance exceptions, open compliance items.

---

### 1. Workforce structure (4 min)

**Route:** Sidebar → **People & HR** → **Employees**  
[https://hcm.raventechgroup.com/dashboard/employees](https://hcm.raventechgroup.com/dashboard/employees)

**Say:**  
*“Every employee has a single record — role, department, bank details, statutory IDs, and allowances. Fleet, warehouse, and customs all roll up to one HR system of record.”*

**Do:**
- Filter or scan by department: **Fleet & Drivers**, **Warehouse**, **Dispatch**, **Customs & Clearing**.
- Open **Moses Okello** — highlight trip/per diem allowances (payroll story later).
- Open **Patricia Achieng** — customs / clearing role.

**Optional:** **Departments** → [https://hcm.raventechgroup.com/dashboard/departments](https://hcm.raventechgroup.com/dashboard/departments)

[https://hcm.raventechgroup.com/dashboard/credentials](https://hcm.raventechgroup.com/dashboard/credentials)

*“In logistics, compliance isn’t optional — driving licences, PSV badges, ADR awareness, forklift certs, and customs agent licences all have expiry dates. Imara tracks them and surfaces what’s expiring before NTSA or KRA becomes a problem.”*

### 3. Time, rota & attendance (5 min)

**Route:** **Rota & scheduling** → [https://hcm.raventechgroup.com/dashboard/rota](https://hcm.raventechgroup.com/dashboard/rota)

**Say:**  
*“Peak season is when logistics breaks — dispatch publishes shift patterns, drivers and warehouse teams get assigned rota, and attendance feeds payroll.”*

**Do:**
- Show published rota for the current month.
- Mention pinned announcement: **“Peak season rota — dispatch & drivers”**.

**Route:** **Attendance** → [https://hcm.raventechgroup.com/dashboard/attendance](https://hcm.raventechgroup.com/dashboard/attendance)

**Do:**
- Show daily summaries (hours, overtime, lateness).
- Point to **exceptions** — disputes before payroll close.

**Optional (if asked about devices):** **Biometric devices** → [https://hcm.raventechgroup.com/dashboard/biometric-devices](https://hcm.raventechgroup.com/dashboard/biometric-devices)

## Refreshing demo data (internal)

If demo data looks stale before an important meeting:

```bash
cd hris-demo
vercel env pull .env.production.local --environment=production --yes
set -a && . ./.env.production.local && set +a
DEMO_MULTI_CONTEXT=true DEMO_UNIFIED_ADMIN_EMAIL=admin@imara.co.ke \
  npx tsx prisma/seed-demo-multi-vertical.ts
```

This resets **all** verticals on the sales instance (SACCO, Stabex, SwiftFreight, healthcare, travel). Allow ~15–20 minutes.
